# SURF-A Library

This page provides a reference for all functions in the SURF-A library. `gtf_to_uorf_db` is the top-level function which organizes the database formatting and export.

## FASTA Utilities

A set of utilities for working with FASTA sequence data.

::: surfa.fasta_utils

## uORF Utilities

A set of functions for finding and formatting uORF regions from FASTA data.

::: surfa.uorf_utils

## JSON Conversion

This library also provides a set of tools for exporting sequences from the database as a JSON structure. This feature is still in development, with JSON output format subject to change.

::: surfa.json_converter
